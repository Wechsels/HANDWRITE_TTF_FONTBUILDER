# 需求

把 Code-03 的 font_build 抽为独立项目 HANDWRITE_TTF_FONTBUILDER，重构并补齐功能。

## 功能

1. 列数调整（行列网格分页，不再单列）
2. 自定义字库（增删改命名）
3. 字帖字体切换（系统已装字体）
4. 字库名称 / 字体名自定义
5. 主题切换（白 / 黑 / 护眼黄，对比色）
6. 数位板检测（缺失弹 Info 提醒）
7. FontForge 检测（缺失弹 Info 提醒）
8. 前端美化（按钮布局 / qss）
9. 重置字库按钮（二次确认）
10. 标点规格：逗号句号 1/4 左下；分号感叹号 1/2 左；省略号破折号满格
11. 小写字母规格：占下半

## 工程

1. 注释精简
2. README 双语（默认中文）
3. 性能 / 安全 + 付费接口预留（OCR / AI 学习）
4. 打包 Release（单文件 exe + Inno Setup 安装包，输出 release/）
